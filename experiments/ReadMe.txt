This directory contains various complete and semi-complete experiment scripts.  Each of them
should work without modification on a current PEBL install.  They are mainly useful for
modifying to create your own experiments.  Short descriptions below.
--------------------------------------------------------------------
cue.pbl  
words.txt:  stimuli for this task

--------------------------------------------------------------------
cue2.pbl
wordsets-new.txt


--------------------------------------------------------------------
IAT.pbl     A version of the "Implicit Attitude Test"
stim/       Directory containing images for test
words-good.txt    Various stimuli files for IAT
words-bad.txt
picnames-good.txt
picnames-bad.txt


--------------------------------------------------------------------
rsvp.pbl    A test of the RSVP (Rapid Serial Visual Presentation) task


--------------------------------------------------------------------
lexicaldecision.pbl  An implementation of the lexical decision task
ld.txt               stores the words and nonwords for the lexical decision task


--------------------------------------------------------------------
masking.pbl          A visual perception 2AFC experiment with briefly-presented stimuli
                     coming from one of four symbols: []().  Masks are
                     generated from these same symbols. 
--------------------------------------------------------------------
letterstrings.pbl    A visual perception 2AFC experiment with briefly-presented
                     letter strings, with the target and foil differing by
                     specified operations (add, subtract, swap, change, etc.)
--------------------------------------------------------------------
--------------------------------------------------------------------
Note:  More experiments/test scripts are available in PEBL\demo
